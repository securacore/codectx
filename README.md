# codectx

A package manager for AI code documentation. Compile, share, and install structured context packages that give AI assistants deep understanding of your codebase conventions, architecture, and workflows.

## Prerequisites

Everything needed to work within this repo is handled by these dependencies.

- [devbox]()
- [just]()

## Getting Started

> List all available commands

```bash
just
```

> Install and setup project.

```bash
just install
```

> Run the CLI.

> [!INFO]
> All arguments passed to this are run within the context of the container.

```bash
just codectx ...
```
